# Time Tracking Dashboard

A responsive and interactive time-tracking dashboard that displays daily, weekly, and monthly activity stats with smooth transitions and hover effects.

Live Demo:https://relaxed-kitsune-788530.netlify.app/

![Time Tracking Dashboard Preview](./preview.jpg)

## Overview

This project is a solution to the [Time tracking dashboard challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/time-tracking-dashboard-UIQ7167Jw).

The dashboard allows users to:

- View their time spent on different activities (Work, Play, Study, Exercise, Social, Self Care)
- Switch between **Daily**, **Weekly**, and **Monthly** timeframes
- See accurate "Last Period" text (Yesterday / Last Week / Last Month)
- Fully responsive design that works perfectly on mobile, tablet, and desktop
- Beautiful hover states and smooth interactions

## Features

- Fully dynamic cards generated using JavaScript and `data.json`
- Clean, semantic HTML5 markup
- Modern CSS Grid & Flexbox layout
- Responsive design with mobile-first approach
- Hover effects on cards and interactive elements
- Active state styling for timeframe buttons
- Accurate period labeling ("Yesterday", "Last Week", "Last Month")

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript (Fetch API, DOM manipulation)
- Google Fonts - [Rubik](https://fonts.google.com/specimen/Rubik)
